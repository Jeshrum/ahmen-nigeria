# Ahmen Nigeria Limited - Business Portal

A modern, glassmorphic business dashboard and landing page for Ahmen Nigeria Limited. Built with high-performance static delivery and premium UI/UX design.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3.4-blue?style=for-the-badge&logo=tailwind-css)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

## 🌟 Features

- **Glassmorphism UI**: Premium visual style with backdrop filters and subtle borders.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Dark Mode Support**: Context-aware styling with smooth transitions.
- **Interactive Components**: Real-time charts and data visualization using Recharts.
- **AI-Powered**: Integrated Google Generative AI for business intelligence.

## 🚀 Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript (ESM)
- **Styling**: Tailwind CSS
- **Visualization**: Recharts, Lucide React
- **PDF Generation**: jsPDF
- **Intelligence**: Google Gemini AI (@google/genai)

## 📂 Project Structure

```text
/
├── assets/           # Compiled JavaScript and CSS assets
├── index.html        # Main entry point
├── netlify.toml      # Deployment configuration for Netlify
└── index.css         # Global styles and custom variables
```

## 🛠️ Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/ahmen-nigeria.git
   ```

2. **Run locally**:
   Since it's a static site, you can use any local server:
   ```bash
   npx serve .
   ```
